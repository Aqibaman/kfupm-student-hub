# KFUPM Student Hub

Prototype full-stack student services app built with Next.js and Tailwind CSS.

## Local Run

1. Install Node.js 16.14+ or newer.
2. Open a terminal in this project folder.
3. Run `npm install`
4. Run `npm run dev`
5. Open `http://localhost:3000`

## Build

- Run `npm run build`
- Run `npm start`

## Notes

- The app uses mock/local demo data.
- Rewards, emergency requests, laundry bookings, and some form actions use browser local storage.
